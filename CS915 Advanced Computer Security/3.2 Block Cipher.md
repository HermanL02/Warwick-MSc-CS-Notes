# PRF and PRP 

# Example
AES:   K × X  →  X  where      K = X = {0,1}$^{128}$
3DES:   K × X  →  X  where      X = {0,1}$^{64}$ ,  K = {0,1}$^{168}$
- A PRP is a PRF where 
1. X = Y
2. It is invertible
