# Block cipher

- Canonical examples:
    

- DES: n=64 bits k=56 bits
    
- 3DES: n=64 bits k=168 bits
    
- AES: n=128 bits k=128, 192, 256 bits
    

# PRF and PRP 

# Example
AES:   K × X  →  X  where      K = X = {0,1}$^{128}$
3DES:   K × X  →  X  where      X = {0,1}$^{64}$ ,  K = {0,1}$^{168}$
- A PRP is a PRF where 
1. X = Y
2. It is invertible
